一、输入与输出

1.1 输入

name = input("what's your name?\n")

Tips: Python可以不用预定义变量的类型

input()会将任何的输入识别为字符串,我们通过调用int()或float()之类的函数(类似于Java中的数据类型类)讲输入转换为希望的数据类型。

例1.1.1:

    x = input("what's x ?")
    y = input("what's y ?")
    z = x + y
    print(z)

输入 x = 1, y = 2, 上述代码输出结果为: 12; 此时 z 的类型为字符串。

例1.1.2:

    x = input("what's x ?")
    y = input("what's y ?")
    z = int(x + y)
    print(z)

输入x = 1, y = 2, 上述代码的输出结果为: 12; 此时 z 的类型为整型。

例1.1.3:

    x = int(input("what's x?"))
    y = int(input("what's y?"))
    z = x + y
    print(z)

输入x = 1, y = 2, 上述代码输出结果为: 3; 此时 z 的类型为整型。

1.2 输出

我们查询Python Built-in Functions来看一看print()的描述。

    print(*objects, sep=' ', end='\n', file=None, flush=False)

我们给出调用该函数的一些实例:

例1.2.1:

    name = "chenyinjie"
    print("Hello, " + name) # 输出: Hello, chenyinjie

    print("Hello,", name) # 输出: Hello, chenyinjie

    print("Hello, ", end = "") # 输出: Hello, chenyinjie(不换行)

    print(name) # 输出: chenyinjie

    print("Hello,",name, end="???") # 输出: Hello, chenyinjie???

    print("Hello,", name, sep="???") # 输出: Hello,???chenyinjie

从上述可知,sep用于控制字符间的插入符号,end控制输出的末尾输出的符号。

针对较长的打印时,我们可以用如下方式进行处理: 例1.2.2:

    name = "chenyinjie"
    print(f"Hello, {name}", end = "") # 输出: Hello, chenyinjie

    print("Hello, {name}") # 输出: Hello, {name}

    print(f"Hello, {name}") # 输出: Hello, chenyinjie

使用print()打印冒号 "" 的方式 例1.2.3:

    print('"friend"') # 输出: "friend"

    print('"friend') # 输出: "friend

    print("Hello, \"friend\"") # 输出: Hello, "friend"

1.3 常见的一些字符串操作

  • str.strip()移除字符串中的前后缀空格
    name = "    chen yin jie    "

    print(f"{name}") # 输出:     chen yin jie     

    name = name.strip() 
    
    print(f"{name}") # 输出:chen yin jie
  • str.capitalize()大写字符串的第一个单词的首字母
    name = "chen yin jie"
